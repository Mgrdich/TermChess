# TermChess — Product Summary

## Vision

A terminal-based chess application for learning and AI development—providing gameplay from casual PvP to challenging RL-powered bots.

## Target Audience

- Chess enthusiasts who prefer CLI tools
- Developers and power users
- Players seeking offline, lightweight chess
- Users interested in AI opponents of varying difficulty

## Core Features

- **Local PvP** — Two-player chess on the same machine
- **Bot Opponents** — Easy, medium, and hard difficulty levels
- **RL Agent** — Custom-trained reinforcement learning bot
- **Terminal Board** — ASCII/Unicode chess board display
- **Move Input** — Standard algebraic notation
- **FEN Support** — Save/load games via FEN strings; start from any position
